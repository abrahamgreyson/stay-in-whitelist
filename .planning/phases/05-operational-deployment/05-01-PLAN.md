    With:
    ```python
    scheduler = BlockingScheduler()

    # Configure misfire handling to prevent silent job skipping
    scheduler.configure(job_defaults={
        'misfire_grace_time': 300,  # 5 minutes grace for 10-minute intervals
        'coalesce': True,           # Combine missed runs into one
        'max_instances': 1          # Prevent overlapping executions
    })

    job = partial(check_and_update_ip, config=config)
    scheduler.add_job(job, 'interval', seconds=config.check_interval)
    ```

    Rationale (per D-01, D-02, D-04):
    - misfire_grace_time=300: 5-minute grace for 600-second intervals (per CONTEXT.md D-02)
    - coalesce=True: Multiple missed runs execute as single run (per RESEARCH.md Pattern 1)
    - max_instances=1: Prevent race conditions in security group updates
  </action>
